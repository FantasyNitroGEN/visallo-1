
# Element Requeue

Element Requeue allows a developer to submit [Vertices](../vertices.md) and [Edges](../edges.md)
to the Graph Property Worker queue. You can perform one of the following actions:

- Requeue Vertex Properties (by Property Name)
    - use this option to enqueue all of the vertex properties in the system with the specified IRI

- Requeue Edges By Label
    - use this option to enqueue all of the edges in the system with the specified IRI

- Requeue Vertices
    - use this option to enqueue all of the vertices in the system

- Requeue Edges
    - use this option to enqueue all of the edges in the system
